# 🧠 Smart Resume Analyzer

> Transform your resume into a powerful career tool 🚀

---

## 🔍 What is this?

**Smart Resume Analyzer** is a web application that analyzes resumes using AI/NLP techniques and helps users improve their resumes with insights, scoring, and recommendations.

---

## ⚡ Highlights

* 📄 Upload PDF Resume
* 🧾 Extract important details automatically
* 📊 Generate Resume Score
* 🎯 Suggest relevant courses
* 🎥 Provide interview & resume tips
* 🔐 Admin dashboard for data tracking

---

## 🧰 Built With

* **Python**
* **Streamlit**
* **SQLite**
* **Pandas**
* **PyResParser**
* **PDFMiner**
* **Plotly**

## 🛠️ Setup Instructions

### 1. Clone Project

```bash id="w6br7r"
git clone https://github.com/YOUR_USERNAME/AI-PROJECTS.git
cd AI-PROJECTS/Resume-Analyzer
```

### 2. Install Dependencies

```bash id="n8hl3s"
pip install -r requirements.txt
```

### 3. Install NLP Model

```bash id="h73q1l"
python -m spacy download en_core_web_sm
```

---

## ▶️ Run the Project

```bash id="gyhlrq"
streamlit run main.py
```

Open:

```
http://localhost:8501
```
